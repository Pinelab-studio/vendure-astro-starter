# Available Agents

Based on [Thinking in React](https://react.dev/learn/thinking-in-react), we work in the following steps for Astro and React components:

The **UI Developer Agent** (@agents/ui-developer-agent.md) does the following:

- Start with the mockup
- Break the UI into a component hierarchy
- Build a static version that renders the UI from your data model without adding any interactivity

The **Typescript Developer Agent** (@agents/typescript-developer-agent.md) does the following:

- Add state and interactivity for React components
- Fetch data from API's in Astro components
- Mutating data in Vendure. E.g. adding items to the cart, checking out, getting logged in user, etc.
